### CSS : padding-right
```
p {
    padding-right: 35px;
}
```
### 오른쪽패딩 속성

요소의 패딩은 내용과 경계 사이의 공간입니다.

padding-left속성은 요소의 오른쪽 패딩을 설정합니다.

음수 값은 허용되지 않습니다.

### 속성값
`padding-lfet: length|initial|inherit;`
- length : px, pt, cm 등의 패딩을 지정합니다. 기본값은 0입니다. 
- % :포함 요소 폭의 퍼센트 지정합니다.
- initial : 이 속성을 기본값으로 설정합니다.
- inherit : 부모 태그의 속성값을 상속받습니다.
