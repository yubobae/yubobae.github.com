### CSS : grid-row-start
```
.item1 {
  grid-row-start: 2;
}
```
### 정의
이 grid-column-start속성은 항목이 시작할 행을 정의합니다.

현재 IE16 버전 이상에서만 지원하는 속성입니다.

### 속성
`grid-row-start: auto|row-line;`

+ auto : 항목은 흐름에 따라 배치됩니다.	
+ column-line : 항목의 표시를 시작할 행을 지정합니다.
