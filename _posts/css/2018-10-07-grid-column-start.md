### CSS : grid-column-start
```
.item1 {
  grid-column-start: 2;
}
```
### 정의
이 grid-column-start속성은 항목이 시작할 행을 정의합니다.

현재 IE16 버전 이상에서만 지원하는 속성입니다.

### 속성
`grid-column-start: auto|column-line;`

+ auto : 항목은 흐름 뒤에 배치됩니다.	
+ column-line : 항목 표시를 시작할 열을 지정합니다.
