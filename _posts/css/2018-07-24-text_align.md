## CSS : text-align
```
div{
  text-aling:left|right|center|justify|initial|inherit;
  }
```
### 텍스트 정렬 속성
이 'text-align'속성은 요소의 텍스트 가로 배열을 지정합니다.
### 속성값
+ left : 왼쪽 정렬 합니다.
+ right : 오른쪽 정렬 합니다.
+ center : 가운데로 정열 합니다.
+ jusrify : 양쪽을 기준으로 정열합니다.
+ initial : 기본값
+ inherit : 상속
