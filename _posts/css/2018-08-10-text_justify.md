## CSS : text-justify 
```
div{
  text-align:justify;
  text-justify:auto|inter-word|inter-character|none|initial|inherit;;
  }
```
### 텍스트 정당화 속성
이 text-justify속성은 text-align이 "justify"로 설정된 경우 텍스트의 양쪽 맞춤 방법을 지정합니다.

### 속성값
+ auto : 기본값
+ inter-word : 
+ inter-character : 
+ none : 
+ initial : 
+ inherit : 상속
