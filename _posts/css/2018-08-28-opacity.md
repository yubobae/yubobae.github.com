### css : opacity

```
div{
  opacity:0.5;
  }
```
### 불투명도 속성

opacity속성은 요소의 불투명도 수준을 설정합니다.

설정값은 0~1사이 값만 적용가능하며,

1은 100%로 0.5는 50%로 0은 0%로로 투명도를 나타냅니다.

또한 opacity속성은 배경과 모든 자식 요소에 투명도를 추가합니다.

- number : 0에서 1까지만 설정 가능하며 값이 클수록 불투명 합니다.
- initial : 기본값을 설정합니다.
- inherit : 부모의 값을 상속받습니다.(기본값)
