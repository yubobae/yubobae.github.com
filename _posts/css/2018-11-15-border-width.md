### CSS :border-width

border-style 속성은 요소의 테두리 너비를 설정합니다.

### 속성
`border-width: medium|thin|thick|length|initial|inherit;`

+ medium : 중간 굵기의 너비를 지정합니다. default값 
+ thin :얇은 굵기의 너비를 가집니다.
+ thick :두꺼운 굵기의 너비를 가집니다.
+ length :오른쪽 테두리의 두께를 정의 할 수 있습니다.
+ initial :기본속성을 가집니다.
+ inherit :속성을 상속 받습니다.
