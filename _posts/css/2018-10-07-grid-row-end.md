### CSS : grid-row-end
```
.item1 {
  grid-row-end: span 3;
}
```
### 정의
이 grid-row-end속성은 항목이 확장 될 행 수를 정의하거나 항목이 끝나는 행 행을 정의합니다

현재 IE16 버전 이상에서만 지원하는 속성입니다.

### 속성
`grid-row-end: auto|row-line|span n;`

+ auto : 항목은 한 열에 걸쳐 있습니다.	
+ span n :  항목이 포함할 행 수를 지정합니다.	
+ column-line :  항목 표시를 종료할 행을 지정합니다.
