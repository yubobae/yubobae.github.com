### CSS : color
```
body {
    color: red;
}

h1 {
    color: #00ff00;
}

p.ex {
    color: rgb(0,0,255);
}
```
### color

이 line-height속성은 선의 높이를 지정합니다.

### 속성 값

`color: color|initial|inherit;`

+ color :텍스트 색상을 지정합니다. 
  - 컬러명 : red , yellow, green 등등..
  - 헥사 코드 : #ffffff
  - RGB컬러,RGB + opacity : rgb(0,0,0) , rgba(0,0,0,0.1)
+ initial :이 속성을 기본값으로 설정합니다.
+ inherit :상위 요소에서 이 속성을 상속합니다.
