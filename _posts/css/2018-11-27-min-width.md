### CSS :min-width

 min-width속성은 요소의 최소 너비를 정의합니다.

콘텐츠가 최소 너비보다 작은 경우 최소 너비가 적용됩니다.

내용이 최소 너비보다 큰 경우이 min-width속성은 아무 효과가 없습니다.


### 속성
`min-width: length|initial|inherit;`

+ length :기본값은 0입니다. 최소 너비를 px, cm 등으로 정의합니다.
+ % :포함 블록의 최소 너비(백분율)를 정의합니다.
+ initial :기본속성을 가집니다.
+ inherit :속성을 상속 받습니다.
