### CSS :min-height

min-height속성은 요소의 최소 높이를 정의합니다.

내용이 최소 높이보다 작 으면 최소 높이가 적용됩니다.

내용이 최소 높이보다 큰 경우에는 min-height속성이 적용되지 않습니다

### 속성
`min-height: length|initial|inherit;`

+ length : 기본값은 0입니다. 최소 높이(px, cm 등)를 정의합니다.
+ % :포함 블록의 최소 높이(백분율)를 정의합니다.
+ initial :기본속성을 가집니다.
+ inherit :속성을 상속 받습니다.
