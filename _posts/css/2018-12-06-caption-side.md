```
#example1 {
    caption-side: bottom;
}

#example2 {
    caption-side: top;
}
```
### CSS :caption-side

caption-side속성은 표 캡션의 위치를 지정합니다.

### 속성
`caption-side: top|bottom|initial|inherit;`

+ top :표 위에 캡션을 놓는다. 기본값
+ bottom :표 아래에 캡션 놓기
+ initial :	 속성을 기본값으로 설정
+ inherit :속성을 상속 받습니다.
