### CSS : grid-row-gap
```
.grid-container {
  grid-row-gap: 50px;
}
```
### 정의
이 grid-row-gap속성은 그리드 레이아웃에서 행 사이의 간격 크기를 정의합니다.

현재 IE16 버전 이상에서만 지원하는 속성입니다.

### 속성
`grid-row-gap: length;`

+ length : px 또는 %와 같은 모든 길이 값이 기본값입니다.
