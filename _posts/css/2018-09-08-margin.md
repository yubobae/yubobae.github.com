### CSS : margin
```
p {
    margin: 35px;
}
```
### 마진속성
속성은 요소의 여백을 설정하며 다음 속성에 대한 속기 속성입니다.

* margin-top
* margin-right
* margin-bottom
* margin-left
음수 값이 허용됩니다..

### 속성값
`margin: length|initial|inherit;`
- length : px, pt, cm 등의 여백을 지정합니다. 기본값은 0입니다.음수값은 허용됩니다.
- % :포함 요소 폭의 여백을 지정합니다.
- auto : 브라우저는 마진을 계산합니다.
- initial : 이 속성을 기본값으로 설정합니다.
- inherit : 부모 태그의 속성값을 상속받습니다.
