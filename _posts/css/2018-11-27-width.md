```
div.a {
    width: auto;
    border: 1px solid black;
}

div.b {
    width: 150px;
    border: 1px solid black; 
}

div.c {
    width: 50%;
    border: 1px solid black; 
}

```
### CSS :width

width 속성은 요소의 너비를 설정합니다.

요소의 너비에는 여백, 테두리 또는 여백이 포함되지 않습니다.


### 속성
`width: auto|value|initial|inherit;`

+ auto : 기본값. 브라우저가 너비를 계산합니다.
+ length : 너비를 px,cm 등으로 정의합니다.
+ % :포함 블록의 폭(백분율)을 정의합니다.
+ initial :기본속성을 가집니다.
+ inherit :속성을 상속 받습니다.
