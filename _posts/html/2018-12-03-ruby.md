### HTML Tag : `<ruby>` 태그
```
<ruby>
漢 <rt> ㄏㄢˋ </rt>

```
### ruby 태그란?

ruby 태그는 루비 주석을 지정합니다.

루비 주석은 주 텍스트에 첨부 된 작은 추가 텍스트로 해당 문자의 발음이나 의미를 나타냅니다. 

이런 종류의 주석은 일본의 출판물에서 자주 사용됩니다.

ruby 태그는 rt 및 / 또는 rp 태그 와 함께 사용하십시오
