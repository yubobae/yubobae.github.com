### HTML Tag : `<header>` 태그
```
 <header>
    <h1>배야월드</h1>
    <h3>프론트엔드 스터디</h3>
    <p>배야월드는 개인적인 스터디 공간입니다.</p>
  </header>
```
### header 태그란?

<header> 요소는 소개 내용 또는 탐색 링크 모음을 위한 컨테이너를 나타냅니다.

<header> 요소는 일반적으로 다음을 포함합니다.

*  하나 이상의 제목 요소(h1~h6)
*  로고 또는 아이콘
*  저자 정보
*  하나의 문서에 여러 개의 <header> 요소를 포함 할 수 있습니다.

<header> 태그는 <footer>, <address> 또는 다른 <header> 요소 내에 배치 할 수 없습니다.
