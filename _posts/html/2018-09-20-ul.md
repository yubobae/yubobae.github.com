### HTML Tag : `<ul>` 태그
```
<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
```
### ul 태그란?

ul태그는 정렬되지 않은 (글 머리 기호) 목록을 정의합니다.

정렬되지 않은 목록을 만들려면 li태그와 함께 ul태그를 사용합니다.

주문 목록을 만들려면 ol태그를 사용합니다.
