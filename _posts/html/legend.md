### HTML Tag : `<legend>` 태그
```
<form>
  <fieldset>
    <legend>Personalia:</legend>
    Name: <input type="text" size="30"><br>
    Email: <input type="text" size="30"><br>
    Date of birth: <input type="text" size="10">
  </fieldset>
</form>
```
### legend 태그란?

legend태그는 fieldset 요소에 대한 캡션을 정의합니다 .
