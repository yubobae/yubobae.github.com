## HTML Tag : `<body>`태그
```
<html>
  <head>
    <title>title 입니당!</title>
  </head>
  <body>
    
  </body>
</html>
```
## body 태그 란?

  body 태그는 문서의 본문을 정의합니다.
  
  body 요소는 텍스트,하이퍼링크,이미지,표,목록 등과 같은 html문서의 모든 내용을 포함합니다.
  
  실제 화면에 나타나는 부분이며 정보공유를 위한 데이터가 들어가는 부분입니다.(컨텐츠의 정보가 들어가는 영역)
