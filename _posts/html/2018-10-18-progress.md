### HTML Tag : `<progress>` 태그
```
<progress value="22" max="100"></progress>

```
### progress 태그란?

progress 태그는 작업 진행률을 나타냅니다.

현재 IE10이상 지원하고 있습니다.

### 속성

+ max : 작업에 필요한 총 작업 수를 지정합니다.
+ value : 완료된 태스크 양을 지정합니다.
