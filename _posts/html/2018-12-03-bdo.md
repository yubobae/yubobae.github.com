### HTML Tag : `<bdo>` 태그
```
<bdo dir="rtl">
This text will go right-to-left.
</bdo>
```
### bdo 태그란?

bdo는 Bi-Directional Override를 나타냅니다.

bdo 태그는 현재 텍스트 방향을 덮어 쓰는 데 사용됩니다.

### 속성

+ dir :필수의 bdo 요소 내부에 있는 텍스트 방향을 지정합니다.(ltr, rtl)
