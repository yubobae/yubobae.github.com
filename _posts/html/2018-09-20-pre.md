### HTML Tag : `<pre>` 태그
```
<pre>
Text in a pre element
is displayed in a fixed-width
font, and it preserves
both      spaces and
line breaks
</pre>
```
### pre 태그란?

pre 태그는 서식이 미리 지정된 텍스트를 정의 합니다.

pre요소의 텍스트는 고정 너비 글꼴로 표시되며 공백과 줄 바꿈을 모두 유지합니다.

즉 html의 태그의 특성도 적용이 되면서 입력한 내용 그대로 화면에 출력을 해줍니다.
